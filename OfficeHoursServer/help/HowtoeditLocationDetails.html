How to edit Location Details

Getting Started ››

To edit Location details, User has to fill in the Branch name, Location Code, and Location Details.  Where Branch name and Location code are required fields which cannot be left blank.  Then User must press “Save.”     If the User does not want to continue, User can press “Cancel” and exit.
