Sign (In/Out) Menu Option

Getting Started ››

In the Sign (In/Out) option provides the User the opportunity to see the available status for the User schedule.  If the User is in the office, a green indicator is on.  If not signed in, a yellow indicator is on.  If the User does not exist in the current schedule, a red indicator is on.  Also, the User can view the date, Branch, Location, Starting Time, Sign In status, the Finishing Time, and the Signed Out status.  If the User is already signed in, the sign in Button is disabled so that the User can sign out easily.  And the link below the Sign (In/out) Panel will navigate the User to My Schedule page in order to make some changes in his schedule.
