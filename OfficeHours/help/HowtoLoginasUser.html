How to Login as User

Getting Started ››

To login as a User the User must Login to the exiting organization and the privileges granted by the Administrator by filling out the required information which includes Organization, Access Code, Username, and Password.  This will result in the opening of the User Control panel which will produce User:  Username in the top right corner of the web page.  After logging into the User Control Panel, the User is allowed to access his schedule and the User can easily modify or add his new schedule.
