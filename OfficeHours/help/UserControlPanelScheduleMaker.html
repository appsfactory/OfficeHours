User Control Panel: Schedule Maker

Getting Started ››

The Schedule Maker is a Schedule builder where the User can make a schedule.  The User can create a Schedule which includes beginning date, ending date, and then selecting day or copying it from the previous month.  There are more options for start time and finishing time with the respective branch and location.
