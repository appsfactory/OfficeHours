How to Login as Administrator

Getting Started ››

If the User has already an existing organization, then the User logs on by going to the Login prompt on the main page.  The User must fill in the Organization, Access Code, Username, and Password and then press “Log in.”  The homepage will appear for Office Hours and in the top right-hand corner will appear either User: Administrator or User: User.  On the homepage you will find the following menu items:  Home, Branches, Locations, Users, Settings, Help, and Log Out if you are the Administrator.  If User opens an existing organization, the Administrator is allowing the User to enter into the User control panel by creating the Organization.
