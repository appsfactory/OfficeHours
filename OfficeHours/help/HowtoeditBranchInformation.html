How to edit Branch Information

Getting Started ››

User as Administrator has to fill in the required information related to Branch such as Branch Name, Address, Phone where Branch name is a required field which cannot be left blank.  Then User must press “Save.”    If the User does not want to continue, User can press “Cancel” and exit.
